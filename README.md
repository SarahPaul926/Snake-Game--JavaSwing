# Snake-Game--JavaSwing

A classic Snake Game built using Java Swing with custom UI and game features.

## Features
- Smooth snake movement
- Apple eating system
- Score tracking
- Pause/Resume (SPACE key)
- Restart game (R key)
- Custom grid background
- Pixel-style snake design

## Technologies Used
- Java
- Java Swing (GUI)
- AWT Graphics

## Controls
- Arrow Keys → Move snake
- SPACE → Pause/Resume
- R → Restart game
